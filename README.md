# Econ Practice in Julia

Not much to see here. Just some code I write to better understand whatever economic concept I happen to be self-studying. I choose Julia as the programming language here because I intend to later use Agents.jl to experiment with economic simulations using agent-based modelling (ABM), but that's for another repo at another time :)

Oh also, I write my code in Quarto documents so that I can render them and display my work using GitHub Pages.
